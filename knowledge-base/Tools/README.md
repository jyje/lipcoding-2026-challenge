## 🎯 빠른 명령어

### 개발
```bash
npm install      # 의존성 설치
npm run build    # 빌드
npm run dev      # 개발 서버 실행
npm test         # 테스트 실행
```

### 배포
```bash
docker build     # Docker 이미지 생성
docker run       # 컨테이너 실행
npm run deploy   # 배포 수행
```
