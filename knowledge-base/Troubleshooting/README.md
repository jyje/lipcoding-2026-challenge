## 🔍 문제 진단 단계

### 1단계: 정보 수집
- 에러 메시지 전체 복사
- 발생 상황 기록 (언제, 무엇을 하다가)
- 로그 파일 확인

### 2단계: 검색
- 이 문서에서 유사한 문제 검색
- 온라인 검색 (에러 메시지로)
- 팀 Slack 검색

### 3단계: 시스템 확인
- 의존성 설치 확인
- 환경 변수 확인
- 버전 호환성 확인

### 4단계: 재현 및 격리
- 최소 예제로 문제 재현
- 최근 변경사항 확인
- 캐시/임시 파일 정리 후 재시도

### 5단계: 도움 요청
- 팀 멤버에게 정보와 함께 문의
- 에이전트 로그 공유
- 재현 가능한 예제 제공
