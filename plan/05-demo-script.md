# 05. Demo Script

## 1) 60초 발표 스크립트

오프닝 (0~10초)

- "LifeOS Insight Coach는 업무, 커리어, 학습 기록을 실행 가능한 행동으로 바꿔주는 개인 생산성 코치입니다."

입력 (10~20초)

- 브레인덤프 예시 입력
- "회의 준비, 이력서 업데이트, 시스템 디자인 학습, 기술 블로그 정리"

분석 결과 (20~40초)

- Top 3 액션 강조
- 각 액션의 이유/예상 시간 확인

실행 계획 (40~50초)

- 30분 플랜 블록 확인
- 완료 토글 1개 시연

마무리 (50~60초)

- "이 앱은 단순 기록이 아니라 지금 무엇을 해야 할지 의사결정을 자동화합니다."

## 2) Q&A 대비 답변 포인트

Q. Copilot SDK를 왜 썼나요?

- "핵심 가치가 자연어 입력을 구조화된 실행 계획으로 바꾸는 과정이라, SDK가 앱의 중심 역할을 합니다."

Q. 일반 할 일 앱과 무엇이 다른가요?

- "업무/커리어/학습 3축을 함께 분석해 우선순위를 제안하고 즉시 실행 블록으로 변환합니다."

Q. 신뢰성은 어떻게 보장하나요?

- "스키마 검증, 재시도, fallback으로 실패해도 최소 결과를 보장합니다."
