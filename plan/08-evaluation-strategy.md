# 08. Evaluation Strategy

## 1) 심사 관점 가정

- 기능 완성도
- AI 활용의 설득력
- 데모 명확성
- 실사용 가능성

## 2) 점수 극대화 전략

1. 기능 완성도
- 핵심 플로우를 짧고 안정적으로 완성
- 실패해도 대체 결과 제공

2. AI 활용 설득력
- 왜 AI가 필요한지 명확히 제시
- 단순 요약이 아니라 의사결정 지원 강조

3. 데모 명확성
- 60초 안에 입력->인사이트->행동 전환 시연
- 시각적으로 Top 3와 이유를 강조

4. 실사용 가능성
- 업무/커리어/학습 모두 아우르는 실제 시나리오 제공

## 3) 감점 방지

- 배포 URL 불안정
- JSON 깨짐으로 UI 렌더 실패
- 기능 과다로 핵심 흐름 실패

## 4) 제출 문구 예시

"LifeOS Insight Coach는 개인의 업무, 커리어, 지식 활동을 통합 분석하여 오늘의 최우선 행동과 실행 계획을 자동 제안하는 Copilot 기반 생산성 앱입니다."
